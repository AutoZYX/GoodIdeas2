# Contributing to Good Ideas 2.0

Thank you for your interest in contributing! This is an open-source parenting book, and every family's perspective makes it richer.

## How to Contribute

### Share Your Scenario (Most Welcome!)

The easiest way to contribute is to share your own parenting scenario. Open a [GitHub Issue](https://github.com/AutoZYX/GoodIdeas2/issues/new) with:

1. **Scenario title**: A short description (e.g., "My kid asked why the sky is blue")
2. **What happened**: The real story — no need to be polished
3. **What you did**: How you responded (honestly — mistakes are welcome!)
4. **What you learned**: Any insight, however small

### Improve Existing Content

- Fix typos or unclear phrasing
- Suggest better AI prompts
- Add translations (especially English versions of Chinese chapters)

### Submit a Pull Request

1. Fork the repository
2. Create a branch (`git checkout -b my-scenario`)
3. Make your changes
4. Submit a PR with a clear description

## Writing Guidelines

- **Be honest.** Real stories > perfect stories.
- **No jargon.** Write so any parent can understand.
- **Respect privacy.** Use pseudonyms for children and other people.

## Code of Conduct

Be kind. We're all figuring out this parenting thing together.
