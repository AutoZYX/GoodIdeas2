# Ch5 AI实践 Prompt

## Prompt 1：AI天空解读者

### Prompt（可直接复制）

```
（拍一张天空中云的照片，上传给AI）
你好，我和孩子正在玩"看云"的游戏。请问这是什么类型的云？
它预示着什么样的天气？请用简单的语言解释，适合5岁孩子理解。
```

### 使用说明

拍照 → 上传 → AI识别云的类型 → 和孩子讨论"它像什么"+"它告诉我们什么"

## Prompt 2：AI露营规划师

### Prompt（可直接复制）

```
你好，请帮我规划一次近郊的家庭烧烤/野餐。
我们是两个家庭，4个大人，3个孩子（5岁、7岁、8岁）。
请生成一份包含以下内容的清单：
1. 食物清单（侧重适合孩子的食物）
2. 工具和装备清单
3. 适合孩子的户外游戏建议（最好能融入"探究"元素）
```

### 核心理念

让AI做"琐碎的规划"，你和孩子专注于"有趣的体验"。这就是AI作为"第三个伙伴"的价值。
