# Ch4 AI实践 Prompt

## 和孩子一起"采访"AI

### Prompt（可直接复制）

```
你好，我和我的孩子正在一起探究一个有趣的文化问题。
我们想知道：为什么中国的公共厕所里，蹲便器比坐便器更常见？
请从以下几个角度来回答，语言要简单，让一个5岁的孩子也能基本听懂：
1. 卫生方面的原因
2. 历史和文化习惯
3. 人体工学（蹲姿 vs 坐姿）
```

### 使用说明

1. 和孩子一起打开AI，念出这个问题
2. AI回答后，问孩子："你觉得哪个原因最有意思？"
3. 如果孩子追问（"那为什么英国没有？"），继续问AI
4. 核心不是"得到正确答案"，而是让孩子体验"提问→探究→讨论"的过程

### 核心理念

当你答不上来的时候，最好的回应不是编一个答案，而是说"**我不知道**，我们一起来问问看"。AI让这句话有了落脚点。
