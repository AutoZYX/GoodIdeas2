# Ch14 AI实践 Prompt

## Prompt 1：用NotebookLM把任何一本书变成"播客"

### 步骤

1. 打开 [NotebookLM](https://notebooklm.google.com)
2. 创建一个新Notebook
3. 上传你想读的书（PDF）或粘贴文章链接
4. 点击 "Audio Overview" → 生成一段两人对话式的音频摘要
5. 戴上耳机，在做饭/散步/带娃时听

> 这是作者真实的工作方式——因为角膜移植需要控制屏幕时间，"听书"成了刚需。

## Prompt 2：创建你自己的"好主意"场景分析

### Prompt（可直接复制）

```
我遇到了一个育儿场景，请帮我用"Good Ideas 2.0"的框架来分析：

【场景描述】：[在这里描述你遇到的具体情况]

请从以下角度分析：
1. Rosen的四个关键词中，哪个最适用？（探究/解读/发明/合作）
2. 自动化等级判断：在这个场景下，家长应该在L0-L5的哪个等级？为什么？
3. 一个可以和孩子一起做的具体活动建议
```

### 使用说明

把方括号里的内容替换成你自己的场景。AI会给你一个结构化的分析。这不是标准答案——它是你思考的起点。

## Prompt 3：为开源项目贡献你的场景

在 GitHub 仓库提交一个 Issue：https://github.com/AutoZYX/GoodIdeas2/issues/new

格式：
```
标题：[场景] 你的场景标题
内容：
- 发生了什么
- 你怎么处理的
- 事后觉得应该怎么处理
- 孩子的年龄
```
