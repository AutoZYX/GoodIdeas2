# Ch1 AI实践 Prompt

## 和孩子一起认识AI

### Prompt（可直接复制）

```
你好，我的孩子今年5岁，他叫[孩子的名字]。
我们刚刚开始一个"好主意"项目——就是在日常生活中一起探索有趣的问题。
[孩子的名字]最近对[孩子感兴趣的东西]非常着迷。
你能用一个5岁孩子能听懂的方式，解释一下[具体问题]吗？
如果可以，请用一个他/她生活中熟悉的东西来做类比。
```

### 使用说明

1. 把方括号里的内容替换成你自己的情况
2. 和孩子一起看AI的回答
3. 问孩子："你觉得它说的对吗？"——这比答案本身更重要
4. 如果孩子提出新问题，继续问AI——让对话自然延伸

### 核心理念

这个Prompt的目的不是"让AI教孩子知识"，而是演示"不懂就问"的行为。
孩子看到你向AI提问，他学到的是：**大人也有不知道的事，不知道不丢人，重要的是去弄明白。**
这正是Rosen说的"I don't know"的力量。
